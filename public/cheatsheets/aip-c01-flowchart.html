Decision Reference
⚡
Exam pattern: ~30% of AIP-C01 questions are service selection scenarios. A question describes a business need and asks which AWS service best fits. This page is your decision map. The key axis is always: how much control vs how much managed simplicity does the scenario need?

Service Deep-Dive Cards
Amazon Bedrock
Core Exam Service
Fully managed API to invoke 20+ Foundation Models. Includes Knowledge Bases (RAG), Agents (multi-step reasoning), Guardrails (safety), Prompt Management, and Model Evaluation.
Use Bedrock when the scenario mentions:
Building a custom GenAI application
RAG pipeline with control over chunking/retrieval
Bedrock Agents for autonomous multi-step tasks
Fine-tuning or model distillation
Safety controls via Guardrails
Custom code required FM included Agents + RAG
Amazon Q Business
No-Code RAG
Fully managed enterprise AI assistant. Connect to 40+ data sources (S3, SharePoint, Confluence, Salesforce). Employees ask questions in natural language — Q retrieves and answers from company docs.
Use Q Business when:
"Zero custom code" or "operational in days"
Internal employee chatbot over company documents
Connecting to SharePoint, Confluence, Salesforce, S3
No ML engineers available
No custom code FM included 40+ connectors
Amazon Kendra
Enterprise Search
Managed intelligent search service combining keyword (BM25) and semantic search. 40+ native connectors to enterprise data sources. Returns ranked search results, not conversational answers.
Use Kendra when:
Search product (results list), not a chatbot
Hybrid keyword + semantic search needed
Enterprise doc search with relevance tuning
Often combined with Bedrock for generation layer
Search only No FM/generation 40+ connectors
Amazon SageMaker
ML Platform
Full ML lifecycle platform: data prep, training, evaluation, deployment, and monitoring of custom models. Required when training your own model architecture or running MLOps pipelines.
Use SageMaker when:
Training a model from scratch or custom architecture
ML engineers need fine-grained training control
MLOps pipelines: feature store, model registry, monitoring
Hosting your own model (not a Bedrock FM)
Custom code required No FM included Full ML control
Amazon Comprehend
NLP Service
Managed NLP for text analysis: sentiment (positive/negative/neutral/mixed), key phrase extraction, named entity recognition, language detection, PII detection, and custom classification.
Use Comprehend when:
Sentiment analysis on customer reviews/feedback
Extract named entities from text at scale
Detect and classify PII in text documents
Route tickets by topic before sending to FM
No generation Text analysis Pre-processing step
Amazon Textract
Document Extraction
Extracts text, tables, forms, key-value pairs, and signatures from scanned documents and PDFs. Essential pre-processing step before feeding document content into a Bedrock RAG pipeline.
Use Textract when:
Scanned PDFs, invoices, medical forms, contracts
Extract structured tables from documents
Pre-process docs before RAG ingestion
"Document extraction" or "OCR" in scenario
Ingestion pre-step Tables + forms No generation
Amazon Rekognition
Computer Vision
Managed computer vision for images and video: object/scene labeling, facial analysis, celebrity recognition, content moderation, text detection (in images), and activity detection in video.
Use Rekognition when:
Image or video analysis needed in the pipeline
Content moderation for user-uploaded media
Multimodal pipeline: image → labels → FM prompt
"Face detection" or "image classification" in scenario
Vision only No text generation Multimodal pre-step
AWS Step Functions
Workflow Orchestration
Visual workflow orchestration with state machines. Coordinates Lambda, Bedrock, SageMaker, and other services with built-in retry, error handling, parallel branches, and human approval steps.
Use Step Functions when:
Multi-step AI pipeline needing retry + error handling
Parallel processing branches required
Coordinate Textract → Comprehend → Bedrock pipeline
"Orchestrate workflow" or "error handling between steps"
Pipeline only Retry built-in No FM

## Exam Weighting

| Domain | Weight | Focus |
|--------|--------|-------|
| Domain 1: Author and Maintain Workflows | ~40% | Triggers, syntax, execution |
| Domain 2: Consume Workflows | ~20% | Marketplace, reusability |
| Domain 3: Author and Maintain Actions | ~25% | Custom actions, publishing |
| Domain 4: Manage GitHub Actions | ~15% | Security, runners, enterprise |

# AI-900: Microsoft Azure AI Fundamentals

AI-900 validates foundational knowledge of **AI concepts**, **machine learning**, and **Azure AI services**. It is designed for technical and non-technical candidates who need to understand when to use common AI workloads and which Azure services support them.

<ExamMeta duration="45 min" questions="40-60" passing="700/1000" cost="Varies by region" level="Fundamentals" provider="Pearson VUE" />

::: warning Retirement Notice
Microsoft lists **AI-900: Microsoft Azure AI Fundamentals** and the related certification as retiring on **June 30, 2026**. This track is expected to be replaced by **AI-901: Azure AI Fundamentals**, with a stronger emphasis on Microsoft Foundry, generative AI apps, and agents.

Use these AI-900 notes if you are targeting the current exam before retirement. If your exam date is after June 2026, verify whether AI-901 has fully replaced AI-900 before booking.
:::

## What the Exam Covers

| Domain | Weight | Focus |
|---|---:|---|
| [Domain 1: AI workloads and considerations](./domain-1.md) | 15-20% | Workload types and responsible AI principles |
| [Domain 2: Machine learning on Azure](./domain-2.md) | 15-20% | Regression, classification, clustering, deep learning, Transformers, Azure ML |
| [Domain 3: Computer vision workloads](./domain-3.md) | 15-20% | Image classification, object detection, OCR, face detection, Azure AI Vision |
| [Domain 4: NLP workloads](./domain-4.md) | 15-20% | Key phrase extraction, entities, sentiment, speech, translation, Azure AI Language and Speech |
| [Domain 5: Generative AI workloads](./domain-5.md) | 20-25% | Generative AI models, responsible GenAI, Azure AI Foundry, Azure OpenAI |
